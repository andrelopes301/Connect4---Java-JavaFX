# Connect4 - Java/JavaFX
 
## Project made for 'Programação Avançada' at ISEC 2020/2021 

### Required
- Java 16
- JavaFX
- Add path to JavaFX on compile
```
--add-modules javafx.controls --module-path "/path/to/javafx/sdk/lib"
```
